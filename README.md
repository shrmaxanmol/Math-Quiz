# Math Quiz 🧮

A fun and interactive Math Quiz web app built with HTML, CSS, and JavaScript.

## Features
- Random math questions (addition, subtraction, multiplication, division)
- Score tracking (+5 for correct, -2 for wrong)
- Animated +5 (green) and -2 (red) feedback effects
- Fully responsive design (mobile, tablet, desktop)
- Smooth hover effects on buttons

## Tech Stack
- HTML
- CSS
- 

## How to Play
1. A random math question will appear
2. Type your answer in the input box
3. Click Submit
4. +5 points for correct answer ✅
5. -2 points for wrong answer ❌
